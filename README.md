# 🛠️ Ferretería API – Sistema de Gestión de Ferretería

**Ferretería API** es una aplicación backend desarrollada con **Java 21** y **Spring Boot**, diseñada para administrar los procesos operativos de una ferretería: gestión de productos, usuarios, boletas, compras y pagos. Esta API utiliza **Firebase Firestore** como base de datos y se encuentra integrada con **Firebase Authentication** para el manejo de usuarios y **Mercado Pago** para pagos en línea.

---

## ✅ Características principales

- Autenticación y autorización con **Firebase Auth** + **JWT**
- CRUD completo de **productos**
- Gestión de **clientes**, **boletas** y **compras**
- Integración con **Mercado Pago** para pagos reales
- Generación de comprobantes de compra
- Encriptación de contraseñas con **BCrypt**
- Protección de rutas por rol (Cliente / Admin)
- Arquitectura en capas para facilitar el mantenimiento

---

## 🧱 Tecnologías utilizadas

- Java 21
- Spring Boot
- Firebase Firestore
- Firebase Authentication
- Mercado Pago Java SDK
- Maven
- Spring Security
- JUnit 5
- Postman (para pruebas manuales)
